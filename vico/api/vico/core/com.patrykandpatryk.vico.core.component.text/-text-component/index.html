TextComponent

open class TextComponent : Padding, Margins
Content copied to clipboard

The component capable of rendering text directly on android.graphics.Canvas. It uses StaticLayout under the hood for text rendering. It supports:

  • Multiple lines of text with automatic line breaking.

  • Text truncation.

  • android.text.Spanned text.

  • Text rotation.

  • Text background with padding. Any Component can be used as text’s background.

  • Text margins.
