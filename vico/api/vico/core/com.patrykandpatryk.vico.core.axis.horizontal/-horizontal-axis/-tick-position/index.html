TickPosition defines the position of ticks. HorizontalAxis.TickPosition.Center allows for using a custom offset and spacing for both ticks and labels.

Parameters

offset

the index at which ticks and labels start to be drawn. The default is 0.

spacing

defines how often ticks should be drawn, where 1 means a tick is drawn for each entry, 2 means a tick is drawn for every second entry, and so on.
